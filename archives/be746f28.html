聚类基本思想

用于处理大量无标记数据集，按照某一种度量方法将数据集分为n个簇，使得簇内样本相似性高，簇间样本相似性低。每个样本仅属于一个簇，每个簇至少有一个样本。

对于给定的类别数目k，首先给出初始划分，通过迭代改变样本和簇的隶属关系，使得每一次改进之后的划分方案都好于之前。

k-Means算法

对于k-Means算法而言k值需预先给定，假定输入n个样本。

  1. 首先随机选择或先验性的给出初始k个类别中心。
  2. 对于每个样本计算到类别中心的距离，将其标记为距离类别中心最近的距离，距离使用欧式距离计算。
  3. 对类别中心进行更新，将其更新为隶属于该类别的所有样本的均值。
  4. 重复2、3步，直至满足终止条件(迭代次数/类中心变化小于某一阈值)。

由于kmeans是将簇中所有点的均值作为新类别中心，因此若簇中含有异常点，将会使得均值严重偏离，因此可以成使用中值，这种聚类方式称为k-Mediods聚类。

记簇中心为μ，每个簇的样本数量为N，此时使用平方误差和为目标函数

所以kmeans算法就是假定了每个簇都服从高斯分布并且方差都相同的前提之下，每次求均值都是沿着梯度方向做了一次梯度下降。k-Mediods聚类则是假定的拉普拉斯分布。

k-Means是初值敏感的，若初始类别中心选定较差，就会把一些本来看似不合适的数据放在同一个类别中，因此对于初始类别中心的选定是非常重要的。初始类别中心相互之间距离越远其效果相对较好，因此可以通过以下方法进行初始类别中心选择。

k-Means是初值敏感的，若初始类别中心选定较差，就会把一些本来看似不合适的数据放在同一个类别中，因此对于初始类别中心的选定是非常重要的。初始类别中心相互之间距离越远其效果相对较好，因此可以通过以下方法进行初始类别中心选择。

  1. 随机选定一个样本作为类别中心
  2. 求其他样本与类别中心的距离，将距离作为权值(假设得到三个样本的距离为0.3,0.5,0.2，则选择第一个样本的概率为0.3，第2个样本的概率为0.5，距离大的样本选中的概率高，距离小的样本选中的概率低，以此类推。)，随机选择一个样本作为第二个类别中心。
  3. 假设当样本与第二个类别中心的距离小于原距离数组的值，则将距离数组中的值进行替换。按照更新之后的距离再将其作为权值随机选择一个类别中心。
  4. 不断更新距离数组并新增类别中心，直至类别中心数量等于k。

该方法相对于k-Means之前随机选择初始类别中心而言更加稳健。这个算法称之为k-Means++。可以做多次k-Means++以找到相对稳定的类别中心。

优点：

  1. 是解决聚类问题的经典算法，简单、快速。
  2. 对处理大数据集，该算法保持可伸缩性和高效性。
  3. 当簇近似为高斯分布时，它的效果较好。

缺点：

  1. 在簇的平均值可被定义的情况下才能使用。
  2. 必须事先给出k个类别并且对初值敏感，对于不同的初始值可能会导致不同结果。
  3. 对噪声和孤立点敏感。
  4. 不适合于发现非凸形状的簇或差别很大的簇。

轮廓系数

聚类的评价指标之一，不需要知道真实值，只需要得到预测值即可对模型进行评估。

计算样本i到同簇其他样本的平均距离为ai,ai越小说明样本i越应该被聚类到该簇。ai称为样本i的簇内不相似度。簇中所有样本的ai均值称为簇不相似度。

计算样本i到其他簇的所有样本的平均距离为bij，称为样本i与簇j的不相似度。定义其为样本i的簇间不相似度。bi=min{bi1,bi2……biK}，bi越大说明样本越不属于其他簇。

根据样本i的簇内不相似度ai和簇间不相似度bi，定义样本i的轮廓系数

s(i)=b(i)−a(i)max{a(i),b(i)}s(i) = \frac{b(i)-a(i)}{max\{a(i),b(i)\}} s(i)=max{a(i),b(i)}b(i)−a(i)​

s越接近1，b就越大于a，说明对应样本i聚类合理，s越接近-1,a就越大于b，说明样本i应该分类在另外的簇，s若趋近于0，则说明样本i在两个簇的边界上。

层次聚类

层次聚类方法对给定的数据集进行层次的分解，直到某种条件满足为止。

凝聚的层次聚类：AGNES算法

一种自底向上的策略，首先将每个对象都作为一个簇，然后不断对这些簇进行合并(将相距最近的两个簇进行合并)，直至满足某个终止条件。

若将最小距离定义为簇间距离，即将两个集合中最近的两个样本的距离作为簇间距离，可能会形成链状结构。

若将最大距离定义为簇间距离，即将两个集合中最远的两个样本的距离作为簇间距离，则异常值会产生较大的影响。

若将平均距离定义为簇间距离，即将两个集合样本间两两距离的平均值或平方和作为簇间距离。

分裂的层次聚类：DIANA算法

一种自顶向下的策略，首先将所有的对象置于一个簇中，然后逐渐细分为越来越小的簇(根据最不相似性对一个簇进行划分)，直至满足某个终止条件。

密度聚类

其思想为，只要样本点的密度大于某阈值，则将该样本添加到最近的簇中，这类算法可以克服基于距离的算法只能发现类圆形的聚类的缺点，可发现任意形状的聚类且对噪声数据不敏感。但计算密度单元的计算复杂度大。

DBSCAN

该算法将簇定义为密度相连的点的最大集合，能够把具有足够高密度的区域划分为簇，并可在有噪声的数据中发现任意形状的簇。

概念

对象的ε-邻域：假设给定对象在半径为ε内的区域，则称该区域为对象的ε-邻域。

核心对象：假设给定数目m，如果一个对象的ε-邻域至少包含m个对象，则称该对象为核心对象。

直接密度可达：假设给定一个对象集合D，如果q为核心对象，对象p在q的ε-邻域内，则称对象p从对象q出发是直接密度可达的(q->p)。

密度可达：假设存在一个对象集合D，如果q1,q2……qn为核心对象，存在一个对象链(qi+1是从qi直接密度可达的)使得q1可以到达qn，则对象qn是从对象q1关于ε和m密度可达的(q1->q2->q3->……->qn)。

密度相连：假设o->p是密度可达，o->q是密度可达，则称p与q时密度相连

簇：一个基于密度的簇是最大的密度相连对象的集合。

噪声：不包含在任何簇中的对象。

算法流程

  1. 给定ε与m
  2. 找到一个点，如果该点的ε-邻域包含多于m个对象，则将该点作为核心对象的簇。
  3. 寻找并合并核心对象直接密度可达的对象
  4. 直至没有新点可以更新簇时，该簇的查询完成
  5. 重复2、3、4直至没有新的未进入簇的点可以作为核心对象时，算法结束。

通过算法流程可以发现每个簇至少包含一个核心对象，非核心对象可以是簇的一部分，构成了簇的边缘，包含过少对象的簇为认为是噪声。

I为除i以外的所有样本，dc为截断距离，ρi为到对象i的距离小于dc的对象的个数。

K近邻均值

取距离样本最近的k个点的距离均值作为密度，并且由于距离越近密度越高，因此取其倒数。

对于样本i而言，存在以下几种情况

  1. ρi大，δi大；ρi大表明样本i周围存在许多样本，δi大表明局部密度比样本i大的样本点距离样本i非常远。因此样本i应该为簇的中心。
  2. ρi小，δi大；ρi小表明样本i周围只存在少量样本，δi大表明局部密度比样本i大的样本点距离样本i非常远。此时样本i应该为噪声。
  3. ρi大，δi小；同理分析，此时样本i属于簇中正常样本。

在确定簇中心后，其他点按照距离已知簇的中心最近进行分类，也可以按照密度可达的方法进行分类。

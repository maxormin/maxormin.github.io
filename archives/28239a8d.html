面向对象

发表于2022-09-11|python
|字数总计:2.2k|阅读时长:7分钟|阅读量:

面向过程

面向过程是开发根据业务逻辑从上到下写代码。

行为(关注点在于怎么做)

把完成某一需求的所有步骤从头到尾逐步实现

根据开发需求，将某些功能独立的代码封装成一个又一个函数

最后完成的代码，就是顺序地调用不同的函数

特点

注重步骤与过程，不注重分工

如果需求复杂，代码会变得很复杂

开发复杂项目，没有固定的开发套路，开发难度大

用例

面向对象

面向对象是将变量与函数绑定到一起，分类进行封装，每个程序只负责分配给自己的分类，减少重复代码。

行为(关注点在于谁来做)

  • 相较于函数，面向对象是更大的封装，根据职责在一个对象中封装多个方法

在温差某一个需求前，首先确定职责——要做的事情(方法)

根据职责确定不同的对象，在对象内部封装不同的方法(多个)

最后完成的代码，就是顺序地调用不同对象的相应方法。

特点

注重对象和职责，不同的对象承担不同的职责

更加适合应对复杂的需求变化，是专门应对复杂项目开发，提供的固定套路

需要在面向过程基础上，再进一步学习

类与对象

类

类是对一群具有相同特征或行为的事物的一个统称，是抽象的。

特征其实就是一个变量，在类中称之为属性

行为其实就是一个函数，在类中称之为方法

类其实是由属性和方法组成的抽象概念

对象

对象是由类创建出来的具体存在。由哪一个类创建出来的对象，就拥有在哪一个类中定义的属性和方法。在开发中应现有类，在类里定义好属性和行为，再根据类来创建对象

类与对象的关系

类是模板，对象是根据类这个模板创建出来的，应该先有类，再有对象。

使用同一个类，能创建出很多对象

类中定义了什么属性和方法，对象中就有什么属性和方法

不同对象对应的属性值也会不同

新建一个对象

在建立一个对象时具体的执行步骤为

  1. 调用__new__方法，渔鸥你过来申请内存空间
  2. 调用__init__方法传入参数，让self指向申请好的那段内存空间，填充数据
  3. 变量s1也指向创建好的内存空间

动态属性

当对象调用没有的属性时，会报错

直接使用等号给一个属性赋值，如果这个属性以前不存在，会给对象添加一个新的属性。如果存在，会修改这属性对应的值。

但是由于该方法较为随意，在某些场景不太合适，因此可以使用__slots__属性进行限制

这时如果在新建对象后通过给属性赋值的方式新增属性，会抛出错误。

魔法方法

是类中的一些特殊方法，其特点为

  1. 不需要手动调用，会在合适的时机自动调用
  2. 这些方法都是使用__开始，使用__结束
  3. 方法名都是系统规定好的，在合适的时机自己调用

__init__方法

在创建对象时，会自动调用这个方法

__str__方法

在init示例的基础上，若自己打印一个对象，显示的是对象的类型以及内存地址

当打印一个对象的时候，会调用这个对象的__str__方法或__repr__方法，若两个方法都写了，会选择__str__方法

__call__方法

对象名() -> 调用这个对象的__call__方法

__setitem__方法

虽然可以通过.__dict__属性获取对象的字典形式。但是在正常情况下是不能直接把一个对象当做字典来使用的。

希望将其作为字典的形式进行赋值时，会调用对象的__setitem__方法

__getitem__方法

与__setitem__方法类似，用于取值。

__eq__方法

is 比较两个对象的内存地址

== 比较值,会调用对象的__eq__方法，获取这个方法的比较结果，本质为num1.__eq__(num2)，获取这个方法的比较结果

在不重写__eq__方法的情况下，将自己编写的同一类下的两个对象使用 is 与 == 进行比较都是False

__eq__方法扩展

*表示写了该方法才可以进行相应运算符的比较，不写就进行比较会报错

  • __ne__方法: 使用 != 运算符会自动调用该方法

  • __gt__方法*: 使用 > 运算符会自动调用该方法

  • __ge__方法*: 使用 >= 运算符会自动调用该方法

  • __lt__方法*: 使用 < 运算符会自动调用该方法

  • __le__方法*: 使用 <= 运算符会自动调用该方法

  • __add__方法*: 使用 + 运算符会自动调用该方法

  • __sub__方法*: 使用 - 运算符会自动调用该方法

  • __mul__方法*: 使用 * 运算符会自动调用该方法

  • __truediv__方法*: 使用 / 运算符会自动调用该方法

  • ……

__str__方法

str()将对象转换成字符串，会自动调用__str__方法。

打印对象也会调用该方法。

int(),float()等同理

内置属性

可以通过dir(s1)方法查看对象s1中的所有属性与行为

  • *代表需要设置后才可以调用

__slots__属性*: 该属性的具体使用方法参考动态属性

__class__属性: 返回该对象所对应的类

__dict__属性：将对象属性和值转化为字典的形式返回

__dir__属性: 等价于dir()函数

__doc__属性: 显示类说明，说明为注释中的内容

__module__属性: 直接运行显示__main__，否则显示模块名
